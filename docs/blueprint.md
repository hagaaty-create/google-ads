# **App Name**: Hagaaty AI Landing

## Core Features:

- Single Page Layout: Implement a clean, single-page design to present Hagaaty and Google Ads information effectively.
- Content Sections: Divide the page into clear sections: About Hagaaty, Google Ads Explanation, and Call to Action.
- Gemini-Powered Assistant: Integrate a Gemini-powered virtual assistant to answer user questions and provide guidance on Hagaaty and Google Ads. The assistant can act as a tool.
- Deployment to Vercel: Configure and deploy the website on Vercel for fast and reliable hosting.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB), evoking trust and clarity, inspired by digital communication and sky.
- Background color: Very light gray (#F0F0F0), providing a clean and modern backdrop.
- Accent color: Light indigo (#6F7ED4) for highlighting important information and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif offering a clean and objective aesthetic.
- Use modern, minimalist icons to represent different sections and features. Make sure they all follow the same aesthetic rules.
- Ensure a responsive layout that adapts to different screen sizes, providing an optimal viewing experience on all devices.
- Incorporate subtle animations for page transitions and interactions to enhance user engagement.
